Getting Started with LangX

Installation

LangX is easy to install and get started with. Follow these steps to set up LangX on your system.

Windows

# Download the installer from the downloads page
# Run the installer and follow the setup wizard
# Verify installation by running:
langx --version

macOS

# Using Homebrew
brew install langx

# Or download from the downloads page
# Verify installation:
langx --version

Linux

# Using package manager
sudo apt-get install langx

# Or download from the downloads page
# Verify installation:
langx --version

Quick Start

Let's write your first LangX program!

Your First Program

Create a file named hello.langx:

print("Hello, World!")

Run it with:

langx hello.langx

Basic Syntax

LangX has a clean and intuitive syntax. Here are some basics:

# Variables
name = "LangX"
version = 1.0

# Functions
func greet(name) {
    return "Hello, " + name
}

# Call function
print(greet(name))
