Getting Started with Tyxar

What is Tyxar?

Tyxar is a Natural-Level Programming Language that is designed to match the natural flow of commands in English, making it:

  1. easy and less daunting for beginners to learn
  2. easy to read and write
  3. faster to colaborate...
To read more about Tyxar, visit the about Tyxar page.

Quick Install
