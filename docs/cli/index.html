Command Line Interface (CLI)

All CLI Commands

Tyxar CLI commands are designed to be easily memorable in order to be used efficiently by both beginners and developers alike.

  • Get CLI help : tyxar help
  • Verify Tyxar Version : tyxar version
  • Run code without building : tyxar run file.txr

    • Inbuilt CLI Helper

    ..... 

    tyxar help 
or
tyxar h
    ...

    Verifying Tyxar Version

    ..... 

    tyxar version
or
tyxar v
    ...

    Running Programs

    Running Tyxar programs... 

    tyxar run(or r) file.txr
tyxar r(or run) file.txr --ast(or -a)
tyxar r(or run) file.txr --jit(or -j)
    ...to run the code without building
