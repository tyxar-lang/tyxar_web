Command Line Interface (CLI)

All CLI Commands

Tyxar CLI commands have specific....

  • Run code without building :tyxar run text.txr
  • Compile/Build code to binary :tyxar build text.txr
  • Transpile code to another high level language, without running or building :tyxar convert text.txr -target:"py"

    • Running Programs

    Running Tyxar programs...tyxar run test.txr... to run the code without building
