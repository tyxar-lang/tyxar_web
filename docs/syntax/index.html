Language Basics

Annotations

Annotations or comments made in code should be written with # on each side of the comment or anotation.

This works for both single lines and more. All text or characters within the #...# would be ignored.

Examples:


# This is a comment #

# This part of the code calls the function
which is then used to give the result#

Variables

Variables are used to store data values. They are declared using the umbrella term VARIABLES: Variables: or VAR: followed by variable declarations.

This provides the relief from scanning through out a file for any declared variables, as all the variables to be used - regardless whether it is empty or not- are declared at the start of a program.

Variable Declaration

Variables declarations in Tyxar are slightly different to many other languages as variable names generally tend to be nouns. For this reason, variable names must always have the KEYWORD: The or the just before the variable names. 

VAR:
The num is 3
the name is "John"

Variable Naming

  • Variable names can contain letters, numbers, and underscores
  • Variable names are case-sensitive
  • Use descriptive names that explain the purpose of the variable
  • Tyxar supports natural language-style variable names

Variable Types

Tyxar supports several data types:

  • Numbers: Integers and floating-point numbers
  • Strings: Text enclosed in double quotes
  • Booleans: true or false values
  • Collections: Arrays and objects

Input/Output

Output

Tyxar uses the keyword write to output data to the screen.

input the name

Data Types

Tyxar is a dynamically typed language, meaning variables can hold values of any type.

Numbers


VAR:
the age is 25
the price is 99.99
the count is 1000

Strings


VAR:
the name is "Tyxar"
the message is "Hello, World!"

Operators

Tyxar supports various operators for performing operations on values.

Arithmetic Operators

VAR:
                        the sum is 2 + 3          # Addition: 5
                        the difference is 10 - 4   # Subtraction: 6
                        the product is 3 * 4       # Multiplication: 12
                        the quotient is 15 / 3     # Division: 5
                        the power is 2 ^ 3         # Exponentiation: 8
                        the remainder is 10 % 3    # Modulo: 1

Comparison Operators

VAR:
                        the isEqual is 5 == 5      # Equal to: true
                        the isNotEqual is 5 != 3   # Not equal to: true
                        the isGreater is 10 > 5    # Greater than: true
                        the isLess is 3 < 5        # Less than: true
                        the isGreaterEqual is 5 >= 5  # Greater than or equal: true
                        the isLessEqual is 3 <= 5     # Less than or equal: true

Logical Operators

VAR:
                        the result1 is true and false   # Logical AND: false
                        the result2 is true or false    # Logical OR: true
                        the result3 is not true         # Logical NOT: false

String Operations

VAR:
                        the greeting is "Hello" + " " + "World"  # Concatenation: "Hello World"
                        the name is "john"
                        the upperName is to adjust the name to CAPS  # Uppercase conversion

Control Flow

Conditionals

Conditional statements allow you to execute different code blocks based on conditions.

If Statements

Use if statements to execute code when a condition is true:

VAR:
                            the age is 18
                            
                            IF the age is greater than 16:
                                write "You can drive"
                            END IF

If-Else Statements

Add an else clause to handle the case when the condition is false:

VAR:
                            the temperature is 25
                            
                            IF the temperature is greater than 20:
                                write "It's warm outside"
                            ELSE:
                                write "It's cool outside"
                            END IF

Else-If Statements

Chain multiple conditions using else-if:

VAR:
                            the score is 85
                            
                            IF the score is greater than 90:
                                write "Grade: A"
                            ELSE IF the score is greater than 80:
                                write "Grade: B"
                            ELSE IF the score is greater than 70:
                                write "Grade: C"
                            ELSE:
                                write "Grade: F"
                            END IF
