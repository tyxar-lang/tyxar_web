Installation

Tyxar is easy to install and get started with. Follow these steps to set up Tyxar on your system.

Windows

# Download the installer from the downloads page
# Run the installer and follow the setup wizard
# Verify installation by running:
tyxar --version

macOS

# Using Homebrew
brew install tyxar

# Or download from the downloads page
# Verify installation:
tyxar --version

Linux

# Using package manager
sudo apt-get install tyxar

# Or download from the downloads page
# Verify installation:
tyxar --version

Quick Start

Let's write your first Tyxar program!

Your First Program

Create a file named hello.tyxar:

print("Hello, World!")

Run it with:

tyxar hello.tyxar

Basic Syntax

Tyxar has a clean and intuitive syntax. Here are some basics:

# Variables
name = "Tyxar"
version = 1.0

# Functions
func greet(name) {
    return "Hello, " + name
}

# Call function
print(greet(name))
